🌱 Smart Plant Disease Detection System  
=====================================

> **AI-powered crop health analysis for the future of smart agriculture**

The **Smart Plant Disease Detection System** is a lightweight yet high-accuracy solution for identifying plant diseases from leaf images. Designed with **real-world agricultural constraints** in mind, this project combines **efficient deep learning** with a **scalable backend** to deliver fast, reliable disease predictions—even in low-resource environments.

Built using **MobileNet** for optimized inference and **Fastapi** for seamless backend integration, this system empowers **farmers, researchers, and agritech developers** to make informed decisions early, reduce crop loss, and adopt precision agriculture with confidence.

---

## 🚜 Why This Project Matters

🌾 Crop diseases are a major threat to food security  
⚡ Early detection = higher yield + lower loss  
📱 Many farmers rely on low-end devices and limited connectivity  

This system is:
- **Fast** → real-time predictions  
- **Lightweight** → runs on mobile & low-power devices  
- **Scalable** → easy to extend to new crops and diseases  
- **Accessible** → simple interface, no ML expertise required  

---

## ✨ Key Features

📸 **Instant Disease Prediction**  
Upload a leaf image and receive accurate disease classification in seconds.

🌿 **Multi-Crop Support**  
Designed to handle multiple crop types and disease classes.

⚙️ **Optimized for Low-Resource Environments**  
MobileNet ensures high accuracy with minimal computational cost.

🌐 **Easy Web & Mobile Integration**  
FastAPI Ensures
- ✅ Data validation via Pydantic
- 🔒 Secure endpoints via dependencies
- ⚙️ Startup/shutdown tasks for DB and models

🧩 **Modular & Customizable Architecture**  
Easily retrain the model, add new classes, or scale the backend.
