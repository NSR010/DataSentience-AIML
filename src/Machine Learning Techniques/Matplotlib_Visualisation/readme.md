# Matplotlib Tutorial 📊  

## 📌 Overview  
This repository contains a Jupyter Notebook (`Matplotlib.ipynb`) that demonstrates the use of **Matplotlib**, one of the most popular Python libraries for data visualization.  
The notebook covers the basics of creating plots, customizing visualizations, and working with different types of charts for better data analysis and storytelling.  

Author - ANU

## 🚀 Features  
- Introduction to **Matplotlib**  
- Creating simple line plots  
- Working with bar, scatter, and pie charts  
- Customizing plots (titles, labels, legends, colors, styles)  
- Subplots and multiple figures  
- Practical examples for data visualization
