# Stock Turnover Ratio Analysis Project

This project analyzes stock market turnover ratios (value traded as % of market capitalization) to understand liquidity and trading behavior over time.

**Contents**
- `run_analysis.py` - simple script to reproduce figures and stats. Saves figures to `output/`.
- `output/` - generated visualizations (time series, histograms, rolling statistics).
- `basic_stats.csv` - descriptive statistics for numeric columns.
- `project_report.txt` - concise objectives, outcomes and business insights.
