# Medical Chatbot

This module contains an NLP-based chatbot designed to provide an initial assessment of medical symptoms through a conversational interface.

## Features

- **Multi-turn Conversation**: The chatbot remembers the context of the conversation to ask follow-up questions.
- **Intent Recognition**: Classifies user input into categories like `report_symptom`, `symptom_severity`, and `emergency`.
- **Symptom Triage**: Provides a basic recommendation based on the reported symptoms and their severity.

## How to Run

1.  **Train the Model**: Before running the chatbot, you must train the intent classifier. This script uses the `data.json` file to create a trained `intent_model.pkl`.
    ```bash
    python train.py
    ```

2.  **Start the Chatbot**: Run the main script to start an interactive session in your terminal.
    ```bash
    python chatbot.py
    ```

## Example Conversation
Medical Chatbot is online! Type 'bye' to exit.
You: I have a headache
Bot: I see. How severe would you say it is (mild, moderate, or severe)?
You: it's mild
Bot: For a mild headache, ensure you are hydrated and have rested. If it persists, please see a doctor.


## Files

- `data.json`: The core dataset containing patterns and responses for different intents.
- `train.py`: The script to train the intent classification model.
- `chatbot.py`: The main application file that runs the interactive chatbot.
- `intent_model.pkl`: The saved model file generated by `train.py`.
- `augment_data.py`: An optional script to create an expanded dataset using synonym repla
